# Ресурси

Перегляд **Ресурси** показує всі ресурси у вашій області доступу та дозволяє створювати й редагувати їх разом із дозволами ролей.

## Типи ресурсів

| Тип | Доступні дозволи |
|-----|-----------------|
| **document** | read (читання), write (запис) |
| **video** | view (перегляд), comment (коментар), stream (потік) |

## Обмеження організацією

Кожен ресурс належить одній організації. Ролям можна надавати дозволи лише на ресурси **тієї ж організації**. Міжорганізаційний доступ організовується через включення ролей.

## Дозволи ролей при створенні/редагуванні

Розділ **Дозволи ролей** показує всі несистемні ролі тієї ж організації. Натисніть будь-який біт для перемикання.

Зміни зберігаються при натисканні **Зберегти** / **Створити** — записуються лише змінені біти.
