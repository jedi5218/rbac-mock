# Розв'язання ефективних дозволів

Перегляд **Дозвіл** обчислює та показує повний набір ефективних дозволів для будь-якого користувача у вашій області доступу.

## Як працює розв'язання

1. Починаємо з усіх ролей, безпосередньо призначених користувачу.
2. Рекурсивно обходимо всі включення ролей (повний граф, з обробкою циклів).
3. Збираємо всі пари `(ресурс, біти дозволів)` з кожної досягнутої ролі.
4. Агрегуємо через `BIT_OR` — користувач отримує об'єднання всіх дозволів.

## Кешування

Результати розв'язання кешуються в пам'яті процесу. Кеш інвалідується:
- **По користувачу** — при зміні його призначень ролей.
- **Глобально** — при зміні будь-якого дозволу або включення ролі.

## Мітки дозволів

| Тип | Біт | Мітка |
|-----|-----|-------|
| document | 1 | read |
| document | 2 | write |
| video | 1 | view |
| video | 2 | comment |
| video | 4 | stream |
